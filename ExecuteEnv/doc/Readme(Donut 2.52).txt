◇このソフトについて
 もう日本一小さくはなくなってしまったタブ付きのMDI型ブラウザです。
 お気に入りはMicrosoft Internet Explorerの物を使用します。
 オープンソースソフトウェアです。

◇配布ファイル
 Donut.exe   実行ファイルです。
 Readme.txt  このファイルです。

◇一度実行したあと生成されるファイル
 Donut.ini   設定を保存しています。削除するとデフォルトの設定になります。
             このファイルのバージョン間の互換性については保証いたしかねますが
             非互換になった場合は報告します。
 Default.dfg 最後の状態を保存しています。

◇動作環境
 Windows95/98 汎用

◇必要なソフト
 Microsoft Internet Explorer 4.0以上

◇インストール方法
 任意のディレクトリに上記のファイルを置いて実行して下さい。

◇アンインストール方法
 Donutのオプションの関連づけのチェックをすべて外してOKを選択した後、
 上記のファイルを削除してください。

◇製作環境
 Microsoft Visual C++ 6.0 ATL3.0/WTL3.1

□Mozillaコントロールの試用について
 未サポートです。ほとんどの機能が有効になりません。
 機能は、DonutというよりMozillaコントロールの実装に依存します。
 別途、Netscape6 PR3以上が必要です。
 コントロールの登録等の作業は自己責任でお願いします。
 最悪の場合は、Donut.iniを削除してください。
 参考 <http://www.iol.ie/~locka/mozilla/mozilla.htm>

□制限
 ○お気に入りメニューを開いたときアクセスする項目の数は、のべ約25900個まで有効です。
 ○お気に入りグループメニューを開いたときアクセスする項目の数は、のべ約2000個まで有効です。
 ○リンクバーに表示できるボタン数は468個まで有効です。

■不具合
○アクティブでないウィンドウを自動更新すると、キー入力ができなくなる。
○ブラウザウィンドウ内の要素へのフォーカスおよびその履歴が失われることがたびたびある。
○IE5.5ではマルチメディア等の制御ができない。

■報告されている不具合
○お気に入りを上書き保存できない。(Win2000)
○小さいアイコンにならない。
○ホイールマウスが効かない。
○起動しない。(IE4)
○Donut.iniが作られない。(Win98/IE5.5)

■ヒント
○空白ウィンドウを開いた後、URLを設定するようなポップアップ広告の場合、
　ポップアップ抑止に「about:blank*」を追加するといいかもしれませんが
　スクリプトエラーが発生する場合があります。

これらの不具合の解決法についての情報がありましたら、ぜひ<mb2@geocities.co.jp>まで。
また、この環境でこれらの不具合が起きる、起きない、についての情報がありましたら、ぜひ<mb2@geocities.co.jp>まで。
