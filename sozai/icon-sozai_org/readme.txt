2008-01-03 +mod

64unDonut.exe で採用しているアイコンです。

unDonut+mod v1.18にてskinでアイコンを設定できるようにしたので、
そのサンプル. 
変更したいskinのフォルダにicon.icoを入れてやってください.
(α付画像があるので、win2k等場合によってはきれい表示できないかも)


作業中、32ビット版と64ビット版を見分けるために仮データほしくて
身内の絵描きさんに雑でいいからとネタふって頼んだら思いのほか
本格的なのがあがってきたので、本採用にしてしまいました。

リサイズをミスってIEより小さい表示になってしまっていますが、
これはこれで、IEと見分けをつけるには（パチモノ臭さが増してて）
丁度いいかな、と。


アイコンデータは、複数サイズをまとめたタイプです。
48x48,32x32,16x16 が入っています。一体化するにあたっては
@icon変換( http://www.towofu.net/soft/aicon.php ) を使いました。
