リンク抽出ツールバーβ3


これは リンク抽出ダイアログを表示する ボタン一個だけを持つ DonutP 用
ツールバープラグインです。
DonutPで対応していない機能の一つ、リンク抽出を行います。



使い方

このDLLを使用するには DonutP5.0β4 が必要です。
DonutP.exe のあるフォルダに Plugin フォルダを作成します。
（おそらく最初からあるはず）
Plugin フォルダの中に LinkExtractBar.dll を 移動 もしくは コピーして
ください。
DonutP を起動して「Donutのオプション」から「プラグイン」タブを選択し、
左側のリストにあるLinkExtractBar.dllにチェックを付け、OK または 適用
を押してオプションダイアログを閉じます。
最後に DonutP を再起動すればツールバーが起動時に配置されるはずです。



リンク抽出ダイアログ

ツールバーのボタンを押すとリンク抽出ダイアログが表示されます。
これは現在開いているタブのリンクを検索し表示するものです。
タブが開いていなかったり、ロードが終わっていないときなどは閉じる以外
のボタンが使用できません。

コンボボックスが３つ並んでいますが、一番上は抽出対象フレームを指定す
るコンボボックスです。
そのページが 複数のフレームに分かれているときは、 この項目から一つを
選択してください。
デフォルトでは全フレームのURLを表示します。

次は抽出対象の拡張子を指定するコンボボックスです。
htm;html のように指定すると絞込みボタンを押したときに、その拡張子を
持つ項目だけが表示されます。
Plugin フォルダ内に LinkExtractBar.dat というファイルを作成し、
jpeg;gif;png
のように書くと読み込むようにしてあります。

最後に非抽出対象の拡張子を指定するコンボボックスです。
ここに書いた拡張子をもつURLは除外されます。

その下のテキストボックスですが、これは検索文字列を入れるところです。

中段の３つ並んだチェックボタンの左側では、検索の方法を指定します。
チェックすると正規表現で文字列比較を行います。
チェックしないと普通の検索です。ワイルドカードも使えます。多分。

真ん中のチェックボックスは選択項目に対する振る舞い方を指定します。
チェックしないと「開く」や「コピー」などのボタンを押したとき、下段リ
ストの全ての項目に対して処理を行いますが、チェックすると選択した項目
にだけ処理を行うようになります。

右のボックスはIMGタグを抽出するか否かです。

これらの条件を用いて URL の絞り込みをするのが 左側中段のボタンになり
ます。
これは下のリストに載っている全項目に対して行われます。


コピーボタンはリストのURLをクリップボードにコピーします。
保存ボタンはURLリストをファイルに書き出します。


Iria,Irvine,DCさくら,FlashGet,NetTrans(port)の各ボタンはそれぞれ対応
するソフトウェアにURLを送ります。
が、実験段階なので正常に動作するかは不明です。




だいぶできて来たとは思うのですが、未熟なものでバグとかも結構あると思
います。気がついたときには報告おねがいします。
その他要望やら感想やら被害報告？とか何でも歓迎します。


β3における変更点
・IMGタグの抽出の有無を選択可能にした
・リスト内の選択項目に対して「開く」「保存」などの処理を実行可能にした
・AREAタグを認識しなかったことによるバグ修正

現在残っている不具合
・URLのコピーを行う際にゴミが入る
（改行コード関係の問題か。未確認）
