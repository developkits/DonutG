■ファイル説明

スナップショット用プラグイン、TmpFavGroup.dllのソースコードです。
サンプルにしては多少コード量が多めなので解説をしたいと思います。

まずこのソースコードはMicrosoftのVC6を使って作成してあります。

メインとなるファイルはTmpFavGroup.cppです。
ここにDLLMainや、プラグインとしての関数が記述されています。

プラグインは CreateExplorerPane 関数が呼ばれるとウィンドウを作成して
本体側にそのハンドルを渡します。
作成するウィンドウはBaseWindow.cppに記述されており、このウィンドウは
一つのツリービューを持ちます。

ツリービューについてはTmpFavTreeView.cppに書いてあります。
このツリービューは 普通のツリービューを拡張して お気に入りグループの
操作を行う関数を多数備えています。

DonutApi.cpp は unDonut が提供する API を 利用するための簡単な機能を
持ちます。
このプラグインでは API を使って お気に入りグループを作成・操作したり
ページを開いたりしています。

ExStyle.h は お気に入りグループ内に書かれてある設定と API 機能で指定
する設定情報を相互変換するためのルーチンを含みます。

MtlProfile.h は DonutP のソースから抜き出したもので Ini ファイル操作
を便利にするクラス CIniSection があります。

PluginInfo.h はプラグインに必要な定義が記述してあります。

ExStyleDialog.cpp は お気に入りグループの項目のセキュリティ設定 等を
行うためのダイアログ機能を持ちます。

TmpFavGroup.def はエクスポートする関数名を記述するファイルです。



■ビルドの方法

StdAfx.hを開き
#import "c:\\ ...
と書かれている行を探します。
このパスを各自の環境にあるunDonut.exeのパスに書き換えてください。
\は2つずつで区切る必要があります。
