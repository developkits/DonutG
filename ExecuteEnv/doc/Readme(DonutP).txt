Donutカスタムしたのです。
名前は、DonutPにします。DonutRの名前のままに
しようと思ったんですが、ややこしいと思ったんでこうなりました。

一応書いておきますが、個人的目的で改変したのもなので、サポート等はあてにしないで下さい(;^_^A ｱｾｱｾ･･･
個人の責任で使用して下さい。（損害が出ても、一切保証しません）
本家等で、発生するバグは、無視する可能性があります。
その時は、諦めた方が無難かも知れません。(;^_^A ｱｾｱｾ･･･

・その他
圧縮ファイルとして一緒に梱包されているHelpファイルは
FILE UP BBSでユーザーの皆様が作成してくれている物です。
(『DonutP.4.2対応版.003』を梱包いたしました。)
最新Verとは異なる事もあり得ますm(_ _)m

Helpファイルを作成して下さいました。
皆様ありがとうございます。
